Brinka iPhone Fullscreen

Alterações:
- Bloqueio de zoom/pinch/double tap zoom
- Sem scroll lateral
- Apenas scroll vertical
- PWA fullscreen/standalone para iPhone
- Inputs com 16px para evitar zoom automático do iOS
- Safe-area para iPhone com notch/dynamic island

Como usar:
1. Envia estes ficheiros para o GitHub Pages.
2. No PC faz CTRL+F5.
3. No iPhone abre no Safari.
4. Partilhar > Adicionar ao ecrã principal.
5. Abre pela app criada no ecrã principal, não pelo Safari normal.
